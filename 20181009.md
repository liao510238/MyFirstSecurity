# openssl

```
root@kali:~# openssl
OpenSSL> sha1 data1.txt
SHA1(data1.txt)= 02d707e871eb91f4a5dc34b6269d3469b987adbf
OpenSSL> sha1 data2.txt      
SHA1(data2.txt)= 589a1c3dd871704c0e506a5b0386cffd932d9ef5
```
